# round-things

A shooter of round things

# How to start

### To run the dev server

```sh
npm install
npm start
```

The app will be accessible from `localhost:8100`.

### To build into `dist/`

```sh
npm install
npm run build
```

The app can be started by loading `index.html` from the `dist/` folder.

### To run all tests

```sh
npm install
npm test
```
